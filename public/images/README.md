# Sports Fest Images

This directory should contain the following images:

## Required Images

### Sport Images (1920x1080 recommended)
- `cricket.jpg` - Cricket field/match image
- `football.jpg` - Football/soccer image
- `basketball.jpg` - Basketball court/game image
- `badminton.jpg` - Badminton court image
- `tabletennis.jpg` - Table tennis image
- `chess.jpg` - Chess board image
- `volleyball.jpg` - Volleyball court image
- `athletics.jpg` - Track and field image

### Hero Section
- `hero-bg.jpg` - Large hero background (1920x1080+)
- `hero-sports.png` - Collage of sports (transparent PNG)

### Logos
- `logo.svg` - Main logo (white version)
- `logo-dark.svg` - Logo for light backgrounds

### Misc
- `banner.jpg` - Social media banner
- `og-image.jpg` - Open Graph image (1200x630)

## Image Guidelines

1. **Format**: Use JPG for photos, PNG for graphics with transparency
2. **Optimization**: Compress images before adding (use tools like TinyPNG)
3. **Naming**: Use lowercase with hyphens (e.g., `hero-bg.jpg`)
4. **Alt text**: Always provide descriptive alt text in components

## Placeholder Images

You can use placeholder services during development:
- https://picsum.photos/
- https://placeholder.com/
- https://unsplash.com/

Example URLs:
```
https://picsum.photos/1920/1080?random=1
https://source.unsplash.com/1920x1080/?cricket
```
